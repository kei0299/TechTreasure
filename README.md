# TechTreasure

## サービス概要
TechTreasureはいろんなプログラミング言語を集めるゲームです。

コインを集めてガチャを回すとプログラミング言語をGETできます。

## 想定されるユーザー層
プログラミング言語に興味がある方

## サービスコンセプト
プログラミング言語はたくさんありますが、どんな言語があるのか
また、歴史や背景など知らないことも多いのではないかと思います。

プログラミング言語について楽しく学ぶことができないかと考えてこのサービスを作ろうと思いました。

## 実装を予定している機能
### MVP
* コレクション一覧
* コレクション詳細
* ガチャページ

### その後の機能
* コインを貯めるためのミニゲーム
  *  神経衰弱
  *  ブロック崩し
  *  メテオパニック
